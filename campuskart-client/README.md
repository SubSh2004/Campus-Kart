# CampusKart Client

Vite + React + TypeScript frontend for CampusKart.

## Setup

Install dependencies:

```bash
cd campuskart-client
npm install
```

Start dev server:

```bash
npm run dev
```

## Built with
- Vite
- React + TypeScript
- Tailwind CSS
- react-router-dom
- axios
- recoil
- socket.io-client
